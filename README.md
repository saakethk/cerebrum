# cerebrum
A python library designed to make getting years of stock data and caching it locally in duckdb format efficient. Abstracts away duckdb functions and such to make it so that people can focus on creating models rather than data.

# Key Dependencies
- https://docs.openbb.co/odp/python
- https://docs.pola.rs/user-guide/getting-started/#installing-polars
- https://duckdb.org
